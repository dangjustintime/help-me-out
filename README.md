# Help Me Out
A social media service where people post small task or odd jobs for people to do
 for money.

## app
[heroku app](https://hidden-ridge-27077.herokuapp.com/)

## Technologies
* HTML
* CSS
* Javascript
* MaterializeCss
* mongoDB
* NPM Packages
    * ejs
    * mongoose
    * express
    * express-session
    * bcrypt
    * method-override

## Unsolved Problems

## User Stories

* Users can do tasks for other people for money
